# Games

A collection of simple games hosted on GitHub Pages.

## Getting Started

### Prerequisites
- Node.js and npm installed

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

The built files will be in the `dist/` directory and ready to be deployed to GitHub Pages.

## Deployment

This project is configured to be deployed on GitHub Pages. The build output in the `dist/` folder will be served from GitHub Pages.
